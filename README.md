# AI Learning System

## 项目定位

面向中学题库学习、作业批改、知识点讲解与个性化学习计划生成的 AI 学习系统。

当前阶段目标：

- 先完成单人开发可维护的项目骨架
- 采用 Spring Boot 作为核心业务后端
- 通过独立 AI 服务承接 OCR、题目讲解、知识点解释、学习计划生成等能力
- 为后续安卓客户端、管理后台和题库数据建设提供统一基础

## 顶层目录

- `docs`：产品、架构、详细设计文档
- `android-client`：安卓客户端工程
- `admin-web`：后台管理前端工程
- `backend-springboot`：Spring Boot 业务后端
- `ai-service`：AI 能力服务
- `infra`：部署、容器、数据库、中间件配置
- `scripts`：初始化、导入、维护脚本
